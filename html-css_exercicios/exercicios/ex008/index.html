Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Utiliza-se a tag DEL.

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Tag INS.
